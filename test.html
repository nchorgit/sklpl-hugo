Γενικές οδηγίες για τα σκεύη αλουμινίου

Σύνθεση

5%-15% ανιονικές επιφανειοδραστικές ουσίες. <5% μη ιονικές επιφανειοδραστικές ουσίες, φωσφονικές ενώσεις, σαπούνι. Επιπλέον περιέχει: Ένζυμα, λαμπρύνουσες ουσίες, συντηρητικά (methylchloroisothiazolinone, methylisothiazolinone, octylisothiazolinone), αρωματικές ουσίες.

Δοσολογία

1 καπάκι : 80ml poly vromika
Μαλακό / Μέτριο νερό 50ml 70ml/80ml
Σκληρό νερό 75ml 100ml
Για πλύση 6-8kg: Προσθέτουμε επιπλέον 30ml

ΠΛΥΣΙΜΟ ΣΤΟ ΧΕΡΙ
Διαλύουμε 30ml απορρυπαντικού σε 10l νερό και στη συνέχεια βάζουμε τα ρούχα

Οδηγίες Προφύλαξης

Κίνδυνος! Προκαλεί σοβαρή οφθαλμική βλάβη. Μακριά από παιδιά. Να φοράτε μέσα ατομικής προστασίας για τα μάτια / το πρόσωπο. ΣΕ ΠΕΡΙΠΤΩΣΗ ΕΠΑΦΗΣ ΜΕ ΤΑ ΜΑΤΙΑ: Ξεπλύνετε προσεκτικά με νερό για αρκετά λεπτά. Εάν υπάρχουν φακοί επαφής, αφαιρέστε τους, εφόσον είναι εύκολο. Συνεχίστε να ξεπλένετε. Συμβουλευτείτε / Επισκεφτείτε αμέσως γιατρό. ΣΕ ΠΕΡΙΠΤΩΣΗ ΚΑΤΑΠΟΣΗΣ: Καλέστε αμέσως το ΚΕΝΤΡΟ ΔΗΛΗΤΗΡΙΑΣΕΩΝ ή ένα γιατρό. Εάν ζητήσετε ιατρική συμβουλή, να έχετε μαζί σας τον περιέκτη του προϊόντος ή την ετικέτα.
Περιέχει: Αλκυλοβενζολοσουλφονικό νάτριο C10-13, άλας νατρίου θειικού εστέρα αιθοξυλιωμένης αλκοόλης C12-14, αιθοξυλιωμένες αλκοόλες C12-15.
Περιέχει methylchloroisothiazolinone και methylisothiazolinone. Μπορεί να προκαλέσει αλλεργική αντίδραση.
Φυλάσσεται σε δροσερό μέρος, μακριά από άμεσο ηλιακό φως και υψηλή θερμοκρασία. Διατηρείται μόνο στην αρχική συσκευασία.
